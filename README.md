# BlockTorch is a simple client-side mod that prevents placing items from the off-hand.

### By default blocks torches and soul torches.
Features:

- Blocks off-hand item placement
- Default torch & soul torch protection
- Item selection via in-game GUI (/bt gui)
- Enable / disable with /bt
- Fully client-side, no server required
- Saves settings to config
- Built-in localization: English and Russian
